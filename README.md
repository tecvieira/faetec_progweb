# Repositório de Aulas - Programador Web FAETEC

Este repositório contém os exercícios e projetos desenvolvidos durante as aulas do curso de **Programador Web** oferecido pela **FAETEC (Fundação de Apoio à Escola Técnica)**.

## Sobre a Instituição

A FAETEC é uma instituição de ensino pública vinculada à Secretaria de Estado de Ciência, Tecnologia e Inovação do Rio de Janeiro. [1] Sua missão é oferecer educação profissional e tecnológica gratuita e de qualidade em diversos níveis, desde a formação inicial e continuada até o ensino superior. [1]

## Sobre o Curso de Programador Web

O curso de Programador Web da FAETEC visa capacitar os alunos a desenvolver, criar e realizar a manutenção de sites e portais para a internet e intranet. A formação abrange as principais tecnologias front-end e back-end, preparando os estudantes para os desafios do mercado de trabalho.

## Conteúdo do Repositório

Aqui você encontrará os materiais de aula, exercícios práticos e projetos propostos pelo professor **André Coutinho**, abrangendo tópicos como HTML, CSS, JavaScript e outras tecnologias relevantes para o desenvolvimento web.
