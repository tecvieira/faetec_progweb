Revisão sobre CSS

Formas de aplicar o CSS

  • Inline

    • Dentro da própria tag

  • Interno

    • Aplicado dentro da tag <style></style>

  • Externo

    • Em um arquivo .css criado em separado, e referênciado através de uma tag <link rel="stylesheet" href="nome_escolhido.css">

Cores, Fontes e Texto

  • color

    • Define a cor do texto

  • background-color

    • Aplica uma cor ao fundo

  • font-size

    • Define um tamanho a fonte

  • font-family

    • Seleciona o tipo de fonte

  • text-align

    • Faz um alinhamento horizontal

  • text-transform

    • Permite transformar todo o texto em minúsculas (lowercase), maiúsculas (uppercase) ou apenas a primeira letra de cada palavra em maiúscula (capitalize).

Box Model (Modelo de Caixas)

Entender como elementos HTML se comportam como caixas com conteúdos internos, bordas e margens.

  • content

    • É uma propriedade usada principalmente com os pseudoelementos ::before e ::after para inserir conteúdo gerado, como texto ou imagens, sem alterar o HTML.

  • padding

    • É uma propriedade que define o espaçamento interno de um elemento, ou seja, a distância entre o conteúdo de um elemento e a sua borda.

  • border

    • É uma propriedade que desenha um contorno (uma linha) em volta de um elemento HTML, permitindo personalizar sua aparência com diferentes estilos (como pontilhado ou contínuo), espessuras e cores.

  • margin

    • Cria um espaçamento externo ao redor de um elemento HTML, definindo a distância entre a borda do elemento e os elementos vizinhos ou o fundo da página, com o objetivo de separar e posicionar os elementos de forma organizada.
